# photo-grid-collage

## 介绍

这是一个基于 vue3 的图片网格拼图组件，可以用于制作 n\*n 图片拼图效果，例如九宫格、81 宫格、114\*114 宫格等。

![image](https://github.com/user-attachments/assets/c5487635-e3bf-4b60-8353-615f67dd5b48)

图片仅用于演示拼图效果，版权归原作者所有。
